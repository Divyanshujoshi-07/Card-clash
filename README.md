#  Card-clash

This project is a **Memory Card Matching Game** built using **HTML, CSS, and JavaScript**. It is a fun and interactive web-based game that helps improve memory and concentration skills.

## 🎮 Game Description
The game consists of a set of cards placed face down, each containing a hidden symbol. The player flips two cards at a time to find matching pairs.

- If the cards match, they remain open.
- If they do not match, they flip back after a short delay.
- The goal is to match all pairs in the least number of moves and time.

## ✨ Features
- 🔄 Card flip animations
- 🎲 Randomized card positions every game
- ⏱️ Timer to track gameplay duration
- 🔢 Move counter
- 🏆 Winning message after completing the game
- 🔁 Restart/Reset game option
- 📱 Responsive design for different devices

## 🛠️ Technologies Used
- HTML (Structure)
- CSS (Styling & Animations)
- JavaScript (Game Logic & Interactivity)

## 💡 Learning Outcomes
- DOM manipulation
- Event handling
- Game logic implementation
- Responsive UI design

## 🚀 How to Play
1. Click on any card to flip it.
2. Select another card to find its match.
3. Match all pairs to win the game.
4. Try to complete the game in fewer moves and less time.
